# **Time Bender**

**A captivating 2D puzzle game where players manipulate time to solve challenging puzzles and overcome obstacles.**

---

## 📖 **Description**

In **Time Bender**, players are immersed in a visually engaging world where time itself is a powerful tool. Solve intricate puzzles by rewinding, pausing, and manipulating objects to progress through diverse levels. Every decision matters, and strategic use of time mechanics is key to overcoming obstacles and unlocking new challenges!

---

## 🎮 **Game Features**
✅ **Time Manipulation Mechanics** – Rewind, pause, and control time to navigate puzzles and levels.

✅ **Engaging Puzzle Design** – Solve increasingly complex puzzles that test logic and creativity.

✅ **Dynamic Interactions** – Interact with objects that respond to time-based actions.

✅ **Immersive Sound & UI** – Designed with an intuitive interface and captivating sound effects.

✅ **WebGL & Desktop Builds** – Play directly in your browser or download the game.

---

## 🚀 **Installation & Setup**

### **Playing via WebGL**
- Simply visit **[Time Bender WebGL Build](https://ucsb-cs148-w25.github.io/pj10-timebendingpuzzle/TimeMachine/mvp/)** to play the game directly in your browser.

### **Running Locally**
1. Clone this repository:
   ```bash
   git clone https://github.com/your-repo/time-bender.git
   ```
2. Open the project in **Unity (Version 2022.3.48f1 or later)**.
3. Click **Play** in Unity Editor or build for your preferred platform.
